Hotel Management System

Requirements:

* User should be able to search hotels by location, brand, distance from airport.
* User should be able to login/register.
* System should support booking different type of rooms. Ex: Standard, deluxe, family suite
* Search room and book the room.
* Allow cancellations, send notification, process refunds and make room available for other users.
* User should be allow to modify booking. Send notification
* Make payments
* Receptionist should be able to add/remove rooms.
* Receptionist should be able to add/modify booking.
* Maintain housekeeping log to keep track of all housekeeping tasks.
* Customer should be able to add room service or food item and should be changed.
* History of all bookings should be recorded.
